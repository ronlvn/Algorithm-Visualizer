# Algorithm Visualizer

אפליקציית דסקטופ שנבנתה ב-WinForms המציגה ויזואליזציה צבעונית בזמן אמת של תהליך המיון באלגוריתמי מיון שונים ומוכרים.

**האלגוריתמים הנתמכים:**
1. מיון בועות (Bubble Sort)
2. מיון מהיר (Quick Sort - פיווט לפי האיבר האחרון)
3. מיון מיזוג (Merge Sort)
4. מיון אקראי (Bogo Sort)

**אפשרויות השליטה בממשק:**
* כפתור ליצירת מערך חדש
* כפתור להתחלת תהליך המיון
* תפריט נפתח לבחירת סוג אלגוריתם המיון 
* שליטה במהירות המיון
* שליטה במספר האיברים שמתמיינים

**להוספה בעתיד:**
* אלגורתמי מיון ליניארים (מיון מניה, מיון דליים ומיון בסיס (Radix))
* טיימר למדידת זמן המיון בms
* כפתור עבור מערך ה worst case
* הוספת הסיבוכיות עבור כל מיון
